Html5 History Api 学习记录

Postado em 2016-09-13   |   Em js

Html5 新增history api, 提供对览器历史记录的访问能力，可以实现无刷新改变浏览器地址栏url。

Html4 history api

  1. length 属性
  2. forward, backward，back 方法
  3. go 方法

Html5 新增 api

pushState(stateObject, title, url)方法

向浏览器历史记录堆栈里添加一条新的记录，调用这个方法传入三个参数，stateObject 可以是对象或 json 字符串；title 目前浏览器暂不支持，可以传入 null; url需为同源的 url。

repalceState(stateObject, title, url)方法

它和history.pushState()方法基本相同，区别是，它不会新添加一条历史记录，而是将当前历史记录替换掉。

popstate事件

点击浏览器的“前进”、“后退”按钮，或是由JavaScript调用的history.back()等导航方法，就会触发此事件。
popstate 事件一般和 pushState/replaceState 方法配合使用，事件处理函数里 event.state 即为 pushState/replaceState时传入的 stateObject。

用途

一个通用的案例是：一个页面，左侧有多个导航按钮，点击导航按钮跳转至相应页面时，只需要替换右边的内容。使用 html5 history 可以实现无刷新跳转页面，更新地址栏 url；记录页面状态，能够使用浏览器前进/后退按钮；可以收藏当前页面或把当前页面分享给别人；有利于搜索引擎抓取
