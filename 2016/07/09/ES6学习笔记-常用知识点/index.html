let & const 命令(block scoped）

let 命令

1. let命令，用来声明变量, 用法类似于var，但是所声明的变量只在let命令所在的代码块内有效。
2. 不存在变量提升, 变量一定要在声明后使用，否则报错。
3. 在同一代码块内，变量不能重复声明

示例代码：

const 命令

1. const 声明一个只读的常量。一旦声明，常量的值就不能改变; 与 let命令一样只在声明所在的块级作用域内有效，不能重复声明
2. const命令声明的常量也是不提升
3. const一旦声明变量，就必须立即初始化，不能留到以后赋值
4. let命令、const命令、class命令声明的全局变量，不属于全局对象的属性

###生成器(Generator）
形式上，Generator函数是一个普通函数，调用方法与普通函数一样，与普通函数不同的是:

  1. function关键字与函数名之间有一个星号；
  2. 函数体内部使用yield语句，定义不同的内部状态；
  3. 调用Generator函数后，该函数不执行，返回一个指向内部状态的指针对象(遍历器对象)，必须调用遍历器对象的next方法，使得指针移向下一个状态。每次调用next方法，内部指针就从函数头部或上一次停下来的地方开始执行，直到遇到下一个yield语句（或return语句）为止。换言之，Generator函数是分段执行的，yield语句是暂停执行的标记，next方法可以恢复执行。
